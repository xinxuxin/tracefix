## Cross-Case Lesson

The most important failure-analysis lesson is that TraceFix needs an independent verifier. A crash-only baseline would accept both `bug_case_06_failure_superficial_fix` and `bug_case_07_failure_ambiguous_behavior`, while TraceFix stops or escalates because the behavioral evidence is insufficient.
